**Focus:** OS concepts, Linux commands, shell scripting, and debugging tools

## üìö Topics Covered

### 1. Basics of OS Concepts (2 hours)
- **Operating System Fundamentals**
  - What is an OS?
  - Kernel vs User Space
  - System Calls
- **Process Management**
  - Process vs Program
  - Process States and Lifecycle
  - Process Scheduling Algorithms
- **Threads**
  - Thread vs Process
  - Multithreading concepts
- **Inter-Process Communication (IPC)**
  - Pipes
  - Message Queues
  - Shared Memory
  - Semaphores
  - Sockets

### 2. Linux Commands & Shell Scripting (2 hours)
- **Essential Commands**
  - File operations: `ls`, `cd`, `pwd`, `cp`, `mv`, `rm`
  - File viewing: `cat`, `less`, `head`, `tail`
  - Text processing: `grep`, `sed`, `awk`
  - System info: `ps`, `top`, `df`, `du`
  - Permissions: `chmod`, `chown`
  - Network: `ping`, `curl`, `wget`
- **Shell Scripting**
  - Bash script basics
  - Variables and arguments
  - Conditionals and loops
  - Functions
  - Automation examples

### 3. Linux Tools (2 hours)
- **Debugging and Monitoring Tools**
  - **WireShark**: Network protocol analyzer
  - **nmap**: Network scanner and security auditing
  - **lsof**: List open files
  - **netstat**: Network statistics
  - **ptrace**: Process trace for debugging
  - **dmesg**: Kernel ring buffer messages
  - **strace**: System call tracer
  - **tcpdump**: Network packet analyzer
- **Log Analysis**
  - Reading system logs
  - Log file locations (/var/log/)
  - Using journalctl
