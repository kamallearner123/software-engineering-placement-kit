# Module 7: Networking Fundamentals

**Duration:** 2 hours  
**Focus:** Basic networking concepts and protocols

## üìã Module Overview

This module introduces essential networking concepts that every software developer should know. You'll learn about network topologies, protocols, the TCP/IP model, IP addressing, and network components like routers and switches.

## üéØ Learning Objectives

- Understand basic networking terminology
- Learn the TCP/IP protocol stack
- Master IP addressing (IPv4 and IPv6)
- Understand network devices and their functions
- Perform hands-on networking commands on Linux

## üìö Topics Covered

### 1. Networking Basics (1 hour)
- **What is a Network?**
  - Definition and purpose
  - Types of networks (LAN, WAN, MAN)
  - Client-Server vs Peer-to-Peer
- **Network Topologies**
  - Bus topology
  - Star topology
  - Ring topology
  - Mesh topology
  - Hybrid topology
- **Protocols**
  - What is a protocol?
  - Common protocols:
    - HTTP/HTTPS
    - FTP
    - SMTP
    - DNS
    - TCP/UDP
- **TCP/IP Model (4 Layers)**
  - Application Layer
  - Transport Layer (TCP/UDP)
  - Internet Layer (IP)
  - Network Access Layer
- **IP Addressing**
  - What is an IP address?
  - IPv4 format and structure
  - IPv6 format and differences
  - Public vs Private IP addresses
  - Subnetting basics
  - CIDR notation

### 2. Network Components (1 hour)
- **Hub**
  - Function and operation
  - When to use
  - Limitations
- **Switch**
  - Function and operation
  - Difference from Hub
  - MAC address table
- **Router**
  - Function and operation
  - Routing tables
  - IP routing
- **Gateway**
  - Function and purpose
  - Default gateway
- **IP Filter / Firewall**
  - Packet filtering
  - Security rules
- **Hands-on Linux Commands**
  - `ping` - Test connectivity
  - `ifconfig` / `ip addr` - View network interfaces
  - `netstat` - Network statistics
  - `traceroute` - Trace packet route
  - `nslookup` / `dig` - DNS lookup
  - `curl` / `wget` - HTTP requests
  - `ssh` - Secure shell
  - `scp` - Secure copy
